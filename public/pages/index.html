Timestamp Microservice API

Coded by Joshua Davidson

Description

The timestamp microservice API takes a single unix timestamp or natural language date and returns a JSON object containg the unix timestamp and the natural language date. If the passed string does not contain a unix timestamp or natural language date then a JSON object containing null values is returned.

Modify the URL above to give it a try!

Example Usage

https://timestamp-microservice-project.herokuapp.com/January 1, 2016
https://timestamp-microservice-project.herokuapp.com/1451606400

Example Output

{ "unix": 1451606400, "natural": January 1, 2016 }
